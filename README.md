# cardapio-ru-sobral
Web scrapper do cardápio semanal do RU da UFC campus Sobral utilizando Python 3

## Instalação
```bash
git clone https://github.com/bot-jonas/cardapio-ru-sobral && \
cd cardapio-ru-sobral && \
pip install -r requirements.txt
```

## Utilização
```bash
python main.py
```
